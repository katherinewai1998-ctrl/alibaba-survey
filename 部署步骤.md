# 🚀 阿里巴巴问卷系统 - 部署步骤

## 由于我无法直接访问您的 GitHub 账号，请按以下步骤操作：

### 方式一：使用 Vercel 部署（推荐，最简单）

#### 第 1 步：推送到 GitHub

1. **访问 GitHub 并创建仓库**
   - 打开 https://github.com
   - 点击右上角 "+" → "New repository"
   - 仓库名：`alibaba-survey`
   - 选择 Public 或 Private
   - 点击 "Create repository"

2. **推送代码**
   
   在终端执行（替换 `YOUR_GITHUB_USERNAME` 为您的 GitHub 用户名）：

```bash
cd /Users/kkkat/.easyclaw/workspace/alibaba-survey

# 配置 Git 用户（首次使用）
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# 添加远程仓库
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/alibaba-survey.git

# 推送
git push -u origin main
```

#### 第 2 步：部署到 Vercel

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击 "Sign Up" 或 "Login"
   - 使用 GitHub 账号登录

2. **导入项目**
   - 点击 "Add New..." → "Project"
   - 找到 `alibaba-survey` 仓库
   - 点击 "Import"

3. **配置项目**
   ```
   Framework Preset: Other
   Root Directory: ./
   Build Command: echo "Building..."
   Output Directory: .
   Install Command: npm install
   ```

4. **添加环境变量**
   点击 "Environment Variables" 添加：
   ```
   ADMIN_PASSWORD = admin123
   NODE_ENV = production
   ```

5. **点击 "Deploy"**
   - 等待部署完成（约 1-2 分钟）
   - 完成后会显示访问地址

#### 第 3 步：获取访问地址

部署完成后，您会看到：
- **域名**: `https://alibaba-survey-xxx.vercel.app`

这就是您的公网访问地址！

### 方式二：使用 Render 部署

#### 第 1 步：推送到 GitHub
（同上）

#### 第 2 步：部署到 Render

1. **访问 Render**
   - 打开 https://render.com
   - 使用 GitHub 登录

2. **创建 Web Service**
   - New + → Web Service
   - 连接 GitHub 仓库
   - 选择 `alibaba-survey`

3. **配置**
   ```
   Build Command: echo "Building..."
   Start Command: node server/index.js
   Environment: Add Variable
     - NODE_ENV = production
     - ADMIN_PASSWORD = admin123
   ```

4. **点击 "Create Web Service"**
