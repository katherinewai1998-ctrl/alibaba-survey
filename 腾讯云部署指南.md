## 🚀 快速部署步骤（5 分钟完成）

### 第 1 步：访问腾讯云 CloudBase

**网址**：https://cloudbase.tencent.com

### 第 2 步：登录

1. 点击 **"登录"**
2. 使用 **微信扫码登录**（最简单）
3. 或 QQ 号登录

### 第 3 步：创建环境

1. 点击 **"创建环境"**
2. 选择 **"免费版"**（个人使用完全免费）
3. 环境名称：`alibaba-survey`
4. 地域：选择 **广州** 或 **上海**（离您最近的）
5. 点击 **"创建"**

等待 1-2 分钟环境创建完成。

### 第 4 步：部署应用

#### 方式一：云托管部署（推荐）

1. 进入环境后，点击左侧 **"云托管"**
2. 点击 **"创建服务"**
3. 配置：
   ```
   服务名称：alibaba-survey
   部署方式：GitHub 部署 或 上传代码
   ```

**如果使用 GitHub 部署**：
- 仓库：`https://github.com/katherinewai1998-ctrl/alibaba-survey`
- 分支：`main`
- 代码路径：`/`

**如果上传代码**：
- 在本地打包：
  ```bash
  cd /Users/kkkat/.easyclaw/workspace/alibaba-survey
  zip -r alibaba-survey.zip . -x "node_modules/*" -x ".git/*"
  ```
- 上传 ZIP 文件

4. 容器配置：
   ```
   CPU：0.25 核
   内存：512 MB
   实例数：1
   ```

5. 启动命令：
   ```
   node server/index.js
   ```

6. 环境变量：
   ```
   NODE_ENV = production
   ADMIN_PASSWORD = admin123
   PORT = 8080
   ```

7. 点击 **"创建"**

#### 方式二：云函数部署（更便宜）

1. 点击左侧 **"云函数"**
2. 点击 **"创建"**
3. 配置：
   ```
   函数名称：alibaba-survey
   运行环境：Node.js 18
   函数代码：上传 ZIP 包或 Git 部署
   执行方法：index.main
   ```
4. 环境变量：
   ```
   NODE_ENV = production
   ADMIN_PASSWORD = admin123
   ```
5. 触发器配置：
   - 添加 **HTTP 触发器**
   - 认证方式：无需认证（公开访问）
6. 点击 **"创建"**

### 第 5 步：获取访问地址

部署完成后：

**云托管**：
- 进入服务详情
- 查看 **"访问域名"**
- 格式：`https://alibaba-survey-xxx.service.tcloudbase.com`

**云函数**：
- 进入函数详情
- 查看 **"触发器 URL"**
- 格式：`https://service-xxx-xxx.gz.apigw.tencentcs.com/release/alibaba-survey`

## 🔧 修改配置

### 修改管理密码

在腾讯云控制台：
1. 进入服务/函数详情
2. 点击 **"环境变量"**
3. 修改 `ADMIN_PASSWORD`
4. 保存并重新部署

### 更新代码

**云托管**：
- 如果使用 GitHub：推送代码后自动部署
- 如果上传 ZIP：重新上传新 ZIP 包

**云函数**：
- 在函数编辑页面更新代码
- 或重新上传 ZIP 包

## 🆘 常见问题

### 1. 部署失败
- 检查启动命令是否正确
- 检查 `package.json` 配置
- 查看部署日志

### 2. 无法访问
- 检查是否配置了 HTTP 触发器
- 检查触发器认证方式（应设为"无需认证"）
- 等待 2-3 分钟部署完成

### 3. 费用过高
- 调整实例配置（降低 CPU/内存）
- 使用云函数更便宜
- 设置实例数为 1
